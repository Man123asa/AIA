Automated n8n workflow for Phishing Mails Detection, Deletion and Alert to the User

This github repository contains the .json file for the workflow created which needs to be imported for running and execution
Since the n8n instance is not shareable please use the copy paste or import method to view and execute the workflow

Requirements for the workflow to work

A Sandbox Gmail Account: A test email inbox to receive the test phishing mail and send out the automated user education alerts.

A VirusTotal API Key
